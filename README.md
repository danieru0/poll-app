# poll app

Create your own polls!
![Poll app gif](https://i.imgur.com/Uv51jw1.gif)

## Features

* Creating polls with max 8 options
* Vote
* Show all polls
* Sort polls by date
* Polls results with chart
* IP checking

## Technologies used

* Express.js
* React
* MongoDB / Mongoose

## How to run locally

1. Download repository
```
$ git clone https://github.com/elosiktv/poll-app.git
```
2. Go to app folder
```
$ cd poll-app
```
3. Install dependencies for server and client
```
$ cd server
$ npm install
$ cd..
$ cd client
$ npm install
```
4. Run MongoDB database
5. Run app
```
$ cd server
$ node server.js / nodemon
$ cd..
$ cd client
$ npm start
```
6.  It's working!
